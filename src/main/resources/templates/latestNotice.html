▶︎ 18 Nov. 2018

v2.5.0 Release

Release URL : https://github.com/scouter-project/scouter/releases/tag/v2.5.0

v2.5.0 Features

  • [server] Zipkin storage feature. - You can now use Scouter as Zipkin server storage. The collected spans can be found in the scouter's xlog view.
  • [server] Support java 9, 10, 11.
  • [webapp] Support java 9, 10, 11.
  • [agent.java] Support java 11.
  • [agent.java] Support Reactor Mono, Flux thread tracing.
  • Support Hystrix thread tracing on a spring cloud feign. (Special Thanks to @redutan)
  • Rabbit mq & kafka producer tracing

v2.5.0 Bug fix

  • [agent.java] fix the active service shows finished service as a zombie services (never finished) with xlog_fully_discard_service options.
  • [agent.java] fix the bug that some xlogs log when the xlog_sampling_only_profile option is enabled.
