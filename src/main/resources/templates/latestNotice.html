▶︎ 28 Sep. 2018

v2.0.0 Release

Release URL : https://github.com/scouter-project/scouter/releases/tag/v2.0.0

v2.0.0 Features

  • [server] Telegraf server feature. - The performance information of various software collected through telegraf input can be integrated and monitored by scouter.
  • [webapp] Interaction counter lookup api added.
  • [webapp] Counter model lookup API added.
  • [agent.java] It is now possible to collect interoperable performance metric information called 'Interaction counter'.
  • [agent.java] It is now possible to collect interoperable performance metric information called 'Interaction counter'.
  • [client] The ability to trace SQL statements using the h2 JDBC driver added.

v2.0.0 Bug fix

  • [agent.java] Fixed a bug where certain requests did not end when the full stack on SQL error option was enabled.
  • [agent.java] Fixed a bug where the '/' character was omitted when parsing SQL literal.
