▶︎ 8 Jun. 2017

v1.7.0 Release

Release URL : https://github.com/scouter-project/scouter/releases/tag/v1.7.0

Features

  • add JDBC driver support of altibase and tibero
  • add performance metric, queuing time
  • Add options for XLog Search limit count and dragging count
  • Traceability for asynchronous processing by producing lambda expressions
  • show detailed description on client's configuration view.
  • change build system and versioning rule. ant -> maven, 0.x -> 1.x

Bug fix

  • Fix database purge bug.
  • Fix the bug that manually redefine classes doesn't work correctly.
