v2.8.1 Release

Release URL : https://github.com/scouter-project/scouter/releases/tag/v2.8.1

v2.8.1 Features

  • [collector] more deilcate adjustable database purging options for removal daily text(dictionary) data, real time counter, visitor counter, tag counter files.
  • [agent.java] xlog sampling judgement follows commencement service's. If the commencement service isn't discarded by sampling rate, the following children services are also not discarded.
  • [agent.java] privide more xlog pattened-sampling configurations.(max 5 types of pattern-set. previous versions provide just one.)
  • [agent.java] add profileSize(Bytes) value onto an XLog property and also into client's filter option.
  • [agent.java] improve profile delivering method.
  • [webapp] add option net_webapp_tcp_client_so_timeout for SO timeout for webapi server connection to collector server.

v2.8.1 MISC

  • [agent.java] jmx counter option set to default false.

v2.8.1 Bug fix

  • [agent.java] fix that it can not be trace async executors on some Jvms.
