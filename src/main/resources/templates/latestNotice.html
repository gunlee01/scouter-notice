▶︎ 21 August. 2017

v1.8.0 Release

Release URL : https://github.com/scouter-project/scouter/releases/tag/v1.8.0

Features

  • Alert script is editable in a scouter client now.
  • Scouter HTTP(Web) api released.
  • JVM metaspace metrics added
  • support postgresql jdbc driver v42+
  • scouter client runs on java 9.

Bug fix

  • [client] fix client hangs when some requests to server are failed.
  • [server] fix that alert script loader can't reload alert script configurations in runtime.
